## ğŸ’¡ Important Notes

### Why Manual Unity Setup?
Unity projects must be created through Unity Hub or Unity Editor. The project files (ProjectSettings/, Packages/, etc.) are generated by Unity and cannot be created programmatically.

### CharacterController vs Rigidbody
The implementation uses `CharacterController` because:
- Provides precise, game-like movement
- No unwanted physics interference
- Standard for first-person games
- Better performance for player control

### Code Architecture
The controller separates concerns:
- **Body rotation**: Horizontal (left/right)
- **Camera rotation**: Vertical (up/down)
- This prevents gimbal lock and feels natural
